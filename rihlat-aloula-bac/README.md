# رحلة الأولى باك

موقع تعليمي عربي متجاوب لمادة الاجتماعيات في السنة الأولى بكالوريا، أعده الأستاذ أحمد بوعمود لمؤسسة الحنان الخاصة.

## البنية المرنة

- `index.html`: هيكل الواجهة فقط.
- `assets/styles.css`: الهوية البصرية والطباعة والاستجابة للهاتف.
- `assets/app.js`: البحث، التصفية، فتح الدروس، التقدم المحلي، والطباعة.
- `data/site.json`: اسم الموقع وبيانات المؤسسة والإصدار.
- `data/lessons.json`: فهرس جميع الدروس وحالتها وكلماتها المفتاحية.
- `data/lessons/*.json`: محتوى كل درس في ملف مستقل.

## إضافة درس جديد

1. أنشئ ملفاً جديداً داخل `data/lessons/` بنفس بنية الدروس الحالية.
2. أضف سجلاً واحداً إلى `data/lessons.json` يحدد العنوان، المادة، الحالة، الكلمات المفتاحية ومسار ملف المحتوى.
3. لا يلزم تعديل `index.html` أو `app.js`.

## التطوير القادم

يمكن إضافة حقول اختيارية لكل درس مثل:

- `documents`: وثائق ونصوص تاريخية.
- `maps`: خرائط وصور.
- `activities`: أنشطة تفاعلية.
- `quiz`: اختبار نهائي.
- `downloads`: ملفات PDF وWord للطباعة.

الموقع مصمم للنشر الساكن على Vercel ويعمل من مسار فرعي دون إعدادات بناء خاصة.
